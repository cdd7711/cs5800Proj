pg224.txt is a sample PDA from the textbook.

Read these PDAs as follows:
1st line is possible states separated by comma without spaces
2nd line is the possible char/terminals separated by comma without spaces
3rd line is possible stack char separated by comma without spaces
4th line is all the accepting states separated by comma without spaces
5th line is the start state

All following lines in the PDA files are transitions. each line is one transition.
start state,terminal,stack head to pop,new state,new stack head pushed
comma separated without spaces
